# Kentlloyd Villanueva Portfolio

A premium one-page personal resume and portfolio website built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion

## Run locally

```bash
npm install
npm run dev
```

## Customize content

Update personal details, links, project cards, certifications, recommendations, and timeline entries in:

- `lib/portfolio-data.ts`

## Project structure

- `app/` App Router entry, layout, and global styles
- `components/portfolio/` Reusable portfolio sections and sidebar
- `components/ui/` Theme toggle and icon primitives
- `components/motion/` Framer Motion reveal helpers
- `public/` Replaceable profile and QR placeholder assets

## Deploy

This project is ready for Vercel deployment:

```bash
npm run build
```
